# Algoritmos y Programación

Ejercicios prácticos de la materia **Algoritmos y Programación** (1º año – Lic. en Matemática Aplicada, FaMAF).

El repositorio cubre conceptos fundamentales de programación, estructuras de control, funciones, recursión, tipos abstractos de datos y algoritmos clásicos.

## Contenidos

### Tarea 1
Traducción a Python de ejercicios incluyendo generación de dibujos.

### Tarea 2
Definición de funciones para dibujar:
- triángulos  
- polígonos regulares  
- una escalera caracol  

Los dibujos se realizan utilizando **ColabTurtle**.

### Tarea 3
Funciones para:
- calcular la suma de los primeros números impares  
- validar el formato de una hora

### Tarea 4
Funciones para:
- determinar si un año es bisiesto  
- calcular los días transcurridos desde el 1/1/1970  
- obtener el día de la semana de una fecha dada

### Tarea 5
Ejercicios con el ciclo `while`:
- cálculo de la parte entera de la raíz cuadrada  
- obtención del primer divisor no trivial de un entero

### Tarea 6
- cálculo de la lista de divisores positivos  
- cálculo del máximo común divisor usando solo sumas y restas

### Tarea 7
- verificación de si una lista está ordenada  
- determinación de si dos listas son permutaciones entre sí

### Tarea 8
Ejercicios de **recursión**:
- cálculo recursivo del máximo común divisor  
- potencia modular

### Tarea 9
Tipos abstractos de datos:
- implementación de la clase `Polinomio`  
- evaluación de polinomios  
- cálculo de raíces racionales de polinomios con coeficientes enteros

### Tarea 10
Ejercicios sobre:
- listas de listas  
- diccionarios  
- manejo de archivos  
- nociones básicas de análisis de datos

## Proyecto Final

### Algoritmo RSA y tests de primalidad

Implementación de los algoritmos necesarios para desarrollar un sistema funcional de cifrado RSA, incluyendo:
- exponenciación modular eficiente  
- tests de primalidad para números grandes  
- generación y uso de claves RSA

Las implementaciones se realizan en **Python**.  
Repositorio con fines académicos.
